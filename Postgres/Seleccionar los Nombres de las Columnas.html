Seleccionar los Nombres de las Columnas de las Tablas en Postgres

Publicado: septiembre 27, 2010 en Base de Datos, Postgres
2

Para obtener los nombres de las Columnas de las Tablas se ejecutará la siguiente instrucción:

select column_name
from information_schema.columns
where table_name = ‘Nombre_Tabla‘;  —  Para obtener los nombres de las columnas de la Tabla ‘Nombre_Tabla‘

Esto es lo que he podido conseguir despues de una ardua búsqueda,  espero les sirva… para lo que mi respecta si funcionó suerte…
